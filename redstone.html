Redstone Javascript

A simple experiment that is getting bigger and better.

  • Written in pure HTML + CSS + JavaScript.
  • No JavaScript framework was used.
  • Valid HTML5 and CSS3.
  • No compatibility with Internet Explorer is expected.
  • Very little compatibility with older browser versions is expected.
  • Tested with Chrome 9, Firefox 3.6 and Opera 11.
  • Written using Vim

Some boring details about the JavaScript code:

  • It defines objects using prototypes, the native JavaScript approach.
  • It uses Array.forEach(), which is already implemented on recent browsers, but I've also provided a JavaScript implementation for browsers that don't have it yet.
  • It uses document.querySelectorAll(). I hope your browser has support for this! (in fact, if it didn't, you wouldn't be reading this text, as these tabs do use this function)
  • It uses JSON.parse() and JSON.stringify(). No fallback is provided if your browser doesn't have this feature.
  • It (mostly) passes JSLint verifications.
